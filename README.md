# NaturalUpdater

Sistem sinkronisasi dan update otomatis untuk seluruh plugin di ekosistem **NaturalSMP**.

## ✨ Fitur Utama
*   **Auto-Sync**: Memastikan seluruh instance server (Bukkit/Velocity) memiliki versi plugin terbaru.
*   **Version Control**: Melacak perubahan dan memastikan kompatibilitas antar modul.
*   **Seamless Update**: Melakukan update tanpa perlu intervensi manual yang berulang.
